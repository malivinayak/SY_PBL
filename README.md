# SY_PBL
Retrieval of student records for admission processing


This Program is of Implication of retrieval of student records for admission processig in C Programing Language
How to access the data : - 1] Student Login
                           2] Admin Login

1] Student Login : 
		  
		  For student portal you don't need any credential for login. Using student credental anyone can but their 
information in system which will be store atomatically. As Your Privacy is First priority no one other than admin can access
your data

2] Admin Login :
		
		 For Accessing the admin portal first you need to login with your credenials If you want to add New user 
then you must enter secert code provided by developer. After Login Admin have two options 
1. Viewing all student data 
2. searching a particular student information. 
In searching a particular student information admin have two option i.e. searching via PRN or searching via keyword.
